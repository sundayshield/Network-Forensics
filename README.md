# Network-Forensics
Three machine investigations fully documented with a dedicated section per machine, all four Brim filters formatted as code blocks, per-machine IOC summary tables, a unified IOC summary table across all three machines, Suricata alert findings, the spoofed user-agent analysis, and identification of all three malware families (Emotet, Redline Stealer, Phorphiex). 

Includes URLhaus and VirusTotal confirmation notes, analyst callout blocks for the 404→200 pattern and the spoofed user-agent, and an 8-entry MITRE ATT&CK mapping.
